# Chatbot Integration (OpenAI)

A Python-based chatbot that integrates OpenAI's API for customer support—handles FAQs, order tracking, and booking inquiries.

---

## 🚀 Features

- Handles customer FAQs
- Tracks orders via API
- Manages booking inquiries
- Built with Flask and OpenAI API
- Lightweight and easy to deploy

---

## 🛠️ Tech Stack

- Python 3.9+
- OpenAI API
- Flask
- Requests

---

```bash
git clone https://github.com/aspacv/chatbot-integration-openai.git
