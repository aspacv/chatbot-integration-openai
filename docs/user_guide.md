# User Guide – Chatbot Integration (OpenAI)

## Overview
A Flask chatbot that integrates OpenAI's API for customer support.

## Setup
1. Clone the repo
2. Install dependencies
3. Add your OpenAI API key to `config.json`
4. Run: `python app.py`
5. Open: `http://localhost:5000`

## Customization
You can modify the system prompt in `app.py` to change bot behavior.
